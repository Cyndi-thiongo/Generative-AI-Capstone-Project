# Generative-AI-Capstone-Project
A simple REST API built with Node.js and Express. Demonstrates a beginner-friendly setup and basic server routes.

Getting Started with Node.js and Express – Building My First REST API
In this project, I explored how to build a simple REST API using Node.js and Express. I chose this technology because APIs are widely used in modern web development and I wanted to understand how backend systems send and receive requests.
The goal of this project is to create a simple server that can respond to requests and return data in JSON format.

Quick Summary of the Technology
Node.js is a JavaScript runtime that allows developers to run JavaScript outside the browser. It is commonly used for building backend services.
Express is a lightweight framework built on top of Node.js that simplifies the process of building web servers and APIs.
These technologies are widely used in real-world applications such as:
•	Social media platforms 
•	E-commerce websites 
•	Mobile app backends 
For example, apps like Uber and Netflix use similar backend technologies to handle requests from users.

Installation Set-up Instructions
Install Node.js- Download Node.js from the official website ( https://nodejs.org )
Create Project Folder
Initialize Node.js – npm init -y
Install Express – npm install express
Create the main file inside Vs code – index.js
Write the server code to the main file
Run the application in the vs code terminal – node index.js
Test in browser - http://localhost:3000
